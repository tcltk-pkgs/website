About

TclTkpkgs is a web interface to search for Tcl/Tk packages and libraries.

Why does tcltkpkgs exist?

Mainly for fun, to explore the Tcl/Tk ecosystem with modern web technologies.

How do I add my package?

Submit a PR to the registry with your package information.
